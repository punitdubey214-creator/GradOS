# GradOS

Graduate Application Operating System

GradOS helps students manage:

* PhD Applications
* Professor Outreach
* Recommendation Letters
* Deadlines
* Notes and SOP Drafts
* Academic Opportunity Search

## Features

* Application Tracker
* Professor Tracker
* Referee Manager
* Deadline Center
* Opportunity Finder
* Personal Dashboard

Built with:

* HTML
* CSS
* JavaScript
* Local Storage
